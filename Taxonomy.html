Taxonomy

This article is about taxonomy in R.

If you have any comments or suggestions for additions or improvements for this article submit an issue , or make some changes and submit a pull request . If you have an issue with one of the packages discussed below, please contact the maintainer of that package.

Taxonomic Data

  • taxize access to 20ish sources of taxonomic data sources
  • taxizedb working with taxonomic database dumps, data at scale
  • ritis Integrated Taxonomic Information System
  • wikitaxa Wikipedia, Wikicommons, Wikspecies, and Wikidata
  • worrms data from WoRMS
  • Taxonstand data from The Plant List
  • Reol data from Encyclopedia of Life

Objects and Manipulating/Parsing Taxonomic Names

  • pegax C++ in R port of gnparser
  • taxa
  • binomen - will be deprecated soon, being wrapped in to taxa

Handling Taxonomic Name Lists

  • defrostR - amphibian taxonomy
  • splister - match species list against a reference list

Working with NCBI and other sequence data

  • microclass Functions for assigning 16S sequence data to a taxonomic level in the tree-of-life for prokaryotes.
  • ncbit making NCBI taxonomic data locally available and searchable as an R object
  • taxonomizr functions for assigning taxonomy to NCBI accession numbers and taxon IDs based on NCBI's accession2taxid and taxdump files

Specialized

  • monographaR functions to facilitate the production of plant taxonomic monographs
  • MonoPhy Explore Monophyly (or Lack of it) of Taxonomic Groups in a Phylogeny
  • taxlist functions to import species lists from Turboveg , a database management system for vegetation data
